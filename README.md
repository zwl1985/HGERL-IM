# HGERL-IM

Implementation of "Learning-based Influence Maximization Framework via Hypergraph Neural Networks and Evolutionary Deep Reinforcement Learning"

Run the code
------------

	python main.py 


Dependency requirement
----------------------

- Python 3.8.8
- NumPy 1.24.4
- PyTorch 2.4.1+cu121
- PyG (PyTorch Geometric) 2.4.0
- PyTorch Scatter 2.0.7
- networkx 3.2.1
